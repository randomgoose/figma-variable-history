# Introduction

**Variable History** is a Figma plugin for managing and synchronizing changes to Figma [variables](https://help.figma.com/hc/en-us/articles/15339657135383-Guide-to-variables-in-Figma).

![Plugin UI](/images/ui.png)

## Key features
- View, discard and commit changes to variables
- Restore pervious versions
- Keep a changelog in Figma
- Export variables of a specific version in CSS for handoff
- Synchronize variables to GitHub repositories, and other tools in the future
