# Sync to GitHub

If you enable GitHub sync, this plugin will synchronize changes when you commit changes.

![sync-to-github](/images/enable-sync.png)

Specifically, the plugin will:
1. compile variables into a single CSS file
2. create a branch in your GitHub repository
3. update the target CSS file on the branch
4. create a pull request

Once it finishes syncing, you can view the pull request in the repository.

![sync-to-github](/images/sync-github.png)
