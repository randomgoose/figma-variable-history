# Enable GitHub sync
This plugin lets you synchronize variable changes with a GitHub repository. To enable this feature, go to <span className="inline-flex items-center gap-1"><Settings2 size={14}/>**Settings**</span> and toggle on **Sync to GitHub Repository**.

![Change types](/images/git-settings.png)

The plugin requires the following information to get access to a repository:
1. Repository Name
2. Username
3. User token (View [this page](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens#deleting-a-personal-access-token) to learn how to create a GitHub user token)
4. File Path (to the CSS file that stores variables)
