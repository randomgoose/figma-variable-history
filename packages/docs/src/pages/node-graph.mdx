# Node Graph

The plugin visualizes variable associations with a node-based UI. To view the node graph, first go to the **Variables** tab.

<br/>

<div>
The number with an arrow icon indicates how many times the variable is aliased. Select a variable, and click the <span className="h-7 inline-flex items-center"><svg xmlns="http://www.w3.org/2000/svg" className="inline w-4 h-4" width="44" height="44" viewBox="0 0 24 24" strokeWidth="1.5" stroke="#000000" fill="none" strokeLinecap="round" strokeLinejoin="round"><path stroke="none" d="M0 0h24v24H0z" fill="none"/><path d="M18 6v6a3 3 0 0 1 -3 3h-10l4 -4m0 8l-4 -4" /></svg></span> icon to view the node graph.
</div>
